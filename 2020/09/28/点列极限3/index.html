зӮ№еҲ—жһҒйҷҗдёүпјҡж•°еҲ—жһҒйҷҗзҡ„дёҠжһҒйҷҗдёҺдёӢжһҒйҷҗпјҢStolzе®ҡзҗҶ

еҸ‘иЎЁдәҺ 2020-09-28 жӣҙж–°дәҺ 2021-01-01 еҲҶзұ»дәҺ ж•°еӯҰ йҳ…иҜ»ж¬Ўж•°пјҡ
жң¬ж–Үеӯ—ж•°пјҡ 6.1k йҳ…иҜ»ж—¶й•ҝ ≈ 6 еҲҶй’ҹ

еҜ№дәҺд»»ж„Ҹз»ҷе®ҡзҡ„ж•°еҲ—$\{a_n\}$пјҢеҰӮжһң$\{a_n\}$жңүз•ҢпјҢз”ұеҲ—зҙ§жҖ§еҸҜзҹҘпјҢеҝ…з„¶еӯҳеңЁж”¶ж•ӣеӯҗеҲ—пјҢеҰӮжһң$\{a_n\}$ж— и§ЈпјҢеҲҷжҖ»еҸҜд»ҘжүҫеҲ°дёҖдёӘеӯҗеҲ—и¶Ӣеҗ‘дәҺ$-\infty$жҲ–$+\infty$гҖӮжҲ‘д»¬жҠҠж•°еҲ—$\{a_n\}$зҡ„ж”¶ж•ӣеӯҗеҲ—зҡ„жһҒйҷҗз§°дёә$\{a_n\}$зҡ„дёҖдёӘжһҒйҷҗзӮ№пјҢиӢҘиҝҳжңүи¶ӢдәҺ$-\infty$жҲ–$+\infty$зҡ„еӯҗеҲ—пјҢд№ҹе°Ҷ$-\infty$жҲ–$+\infty$з§°дёә$\{a_n\}$зҡ„дёҖдёӘжһҒйҷҗзӮ№гҖӮ

е®ҡд№ү1пјҡдёҠжһҒйҷҗдёҺдёӢжһҒйҷҗ

и®ҫ$\{a_n\}$жҳҜдёҖдёӘж•°еҲ—пјҢ$E$жҳҜз”ұ$\{a_n\}$зҡ„е…ЁйғЁжһҒйҷҗзӮ№жһ„жҲҗзҡ„йӣҶеҗҲпјҢи®°

еҲҷ$a^{*}$е’Ң$a_{*}$еҲҶеҲ«з§°дёәж•°еҲ—$\{a_n\}$зҡ„дёҠжһҒйҷҗе’ҢдёӢжһҒйҷҗпјҢи®°дёә

е®ҡзҗҶ1

и®ҫ$\{a_n\}$жҳҜдёҖдёӘж•°еҲ—пјҢ$E$дёҺ$a^*$зҡ„еҗ«д№үдёҺе®ҡд№ү1дёӯдёҖж ·пјҢйӮЈд№Ҳ
пјҲ1пјү$a^* \in E$;

пјҲ2пјүиӢҘ$x>a^*$пјҢеҲҷеӯҳеңЁ$N>\mathbb{N^{*}}$пјҢеҪ“$n \ge N$ж—¶пјҢжңү$a_n < x$;

пјҲ3пјү$a^*$жҳҜж»Ўи¶іеүҚдёӨжқЎжҖ§иҙЁзҡ„е”ҜдёҖзҡ„ж•°гҖӮеҚіз”ұпјҲ1пјүпјҲ2пјүд№ҹеҸҜд»Ҙе®ҡзҗҶдёҠжһҒйҷҗгҖӮ

иҜҒпјҡпјҲ1пјүиӢҘ$a^*=+\infty$пјҢеҲҷ$E$ж— дёҠз•ҢпјҢеҸҜзҹҘ$\{a_n\}$д№ҹжІЎжңүдёҠз•ҢпјҢжүҖд»Ҙеҝ…з„¶еҸҜд»Ҙд»Һ$\{a_n\}$дёӯйҖүеҸ–дёҖдёӘи¶ӢдәҺ$+\infty$зҡ„еӯҗеҲ—пјҢеҚі$a^* \in E$пјҢ
еҰӮжһң$a^*=-\infty$пјҢеҲҷ$E={-\infty}$пјҢжҳҫз„¶з”ұ$a^* \in E$
еҰӮжһң$a^*$жҳҜдёҖдёӘжңүйҷҗж•°пјҢз”ұдәҺ$a^*=\sup E$пјҢд»ҺиҖҢеҝ…еӯҳеңЁ$l_1\in E$пјҢдҪҝеҫ—

еҸҲз”ұдәҺ$l_1$жҳҜ$\{a_n\}$жҹҗдёӘеӯҗеҲ—зҡ„жһҒйҷҗпјҢеҲҷдёҖе®ҡеӯҳеңЁжӯЈж•ҙж•°$k_1$пјҢдҪҝеҫ—

еҗҢзҗҶпјҢеӯҳеңЁ$l_2 \in E$пјҢдҪҝеҫ—

$l_2$жҳҜ$\{a_n\}$зҡ„жҹҗдёӘеӯҗеҲ—зҡ„жһҒйҷҗпјҢеҲҷдёҖе®ҡеӯҳеңЁжӯЈж•ҙж•°$k_2 > k_1$пјҢдҪҝеҫ—

еҪ’зәіеҸҜеҫ—пјҢеҜ№д»»дҪ•$n \in \mathbb{N^*}$пјҢеӯҳеңЁ$k_n$пјҢж»Ўи¶і$k_n>k_{n-1}>\cdots>k_1$пјҢдҪҝеҫ—

жүҖд»Ҙжңү$\lim \limits_{n \to \infty}a_{k_n} = a^*$пјҢеҚіжүҫеҲ°дәҶ$\{a_n\}$зҡ„дёҖдёӘеӯҗеҲ—ж”¶ж•ӣдәҺ$a^*$пјҢжүҖд»Ҙ$a^* \in E$гҖӮ

пјҲ2пјүеҸҚиҜҒжі•гҖӮеҒҮи®ҫеӯҳеңЁж— з©·еӨҡдёӘ$n$пјҢж»Ўи¶і$a_n \ge x$пјҢеҲҷеӯҳеңЁдёҖдёӘ$y \in E$пјҢдҪҝеҫ—$y \ge x > a^*$пјҢиҝҷдёҺ$a^*$жҳҜ$E$зҡ„дёҠзЎ®з•ҢзҹӣзӣҫгҖӮ

пјҲ3пјүеҸҚиҜҒжі•гҖӮеҒҮи®ҫеӯҳеңЁдёӨдёӘдёҚеҗҢзҡ„ж•°$pпјҢq$еҗҢж—¶ж»Ўи¶іпјҲ1пјүпјҲ2пјүпјҢдёҚеҰЁи®ҫ$p<q$пјҢеҸ–$x$пјҢдҪҝеҫ—$p<x<q$пјҢз”ұпјҲ2пјүеҸҜзҹҘпјҢеӯҳеңЁ$N \in \mathbb{N^*}$пјҢдҪҝеҫ—еҪ“$n \ge N$ж—¶пјҢжңү$a_n < x$пјҢжҳ“зҹҘ$E$дёӯзҡ„зӮ№дёҚеҸҜиғҪи¶…иҝҮ$x$пјҢиҝҷжҳҜ$q$дёҚиғҪж»Ўи¶іпјҲ1пјүгҖӮ

Q.E.D.

е®ҡзҗҶ2

и®ҫ$\{a_n\}$жҳҜдёҖдёӘж•°еҲ—пјҢ$E$дёҺ$a_*$зҡ„еҗ«д№үдёҺе®ҡд№ү1дёӯдёҖж ·пјҢйӮЈд№Ҳ
пјҲ1пјү$a_* \in E$;

пјҲ2пјүиӢҘ$x < a_*$пјҢеҲҷеӯҳеңЁ$N>\mathbb{N^{*}}$пјҢеҪ“$n \ge N$ж—¶пјҢжңү$a_n > x$;

пјҲ3пјү$a_*$жҳҜж»Ўи¶іеүҚдёӨжқЎжҖ§иҙЁзҡ„е”ҜдёҖзҡ„ж•°гҖӮеҚіз”ұпјҲ1пјүпјҲ2пјүд№ҹеҸҜд»Ҙе®ҡзҗҶдёӢжһҒйҷҗгҖӮ

иҜҒжі•дёҺе®ҡзҗҶ1зҡ„иҜҒжҳҺж–№жі•еҮ д№ҺдёҖж ·пјҲд»Һз•ҘпјүгҖӮ

е®ҡзҗҶ3

и®ҫ$\{a_n\},\{b_n\}$жҳҜдёӨдёӘж•°еҲ—пјҢ

пјҲ1пјү$\liminf \limits_{n \to \infty}a_n \le \limsup \limits_{n \to \infty}a_n$

пјҲ2пјү$\lim \limits_{n \to \infty}a_n = a$еҪ“дё”д»…еҪ“$\liminf \limits_{n \to \infty}a_n = \limsup \limits_{n \to \infty}a_n = a$

пјҲ3пјүиӢҘ$N$жҳҜжҹҗдёӘжӯЈж•ҙж•°пјҢеҪ“$n>N$ж—¶пјҢжңү$a_n \le b_n$пјҢйӮЈд№Ҳ

иҜҒпјҡпјҲ1пјүпјҲ2пјүжҳҫз„¶пјҢдёӢйқўеҸӘиҜҒпјҲ3пјүзҡ„еҸіиҫ№зӯүејҸпјҢе·Ұиҫ№зӯүејҸеҸҜд»Ҙзӣёдјјзҡ„иҜҒжҳҺгҖӮи®°

иӢҘ$b^*=+\infty$пјҢдёҚзӯүејҸиҮӘз„¶жҲҗз«ӢпјӣеҪ“$a^*=+\infty$пјҢеӯҳеңЁеӯҗеҲ—$a_{k_n} \to +\infty (n \to \infty)$пјҢиҝҷж—¶д№ҹжңү$b_{k_n} \to +\infty (n \to \infty)$пјҢд»ҺиҖҢжңү$b^* = +\infty$пјҢдёҚзӯүејҸзӯүеҸ·жҲҗз«ӢпјӣеҗҢж ·иӢҘ$a^*$дёҺ$b^*$дёӯжңүдёҖдёӘдёә$-\infty$ж—¶пјҢз»“и®әд№ҹжҲҗз«ӢгҖӮзҺ°еңЁи®ҫ$a^*$дёҺ$b^*$йғҪжҳҜжңүйҷҗж•°пјҢйҮҮз”ЁеҸҚиҜҒжі•пјҢеҒҮи®ҫ$b^* < a^*$пјҢеҸ–$x$пјҢдҪҝеҫ—$b^* < x < a^*$пјҢз”ұе®ҡзҗҶ1пјҲ2пјүеҸҜзҹҘпјҢеӯҳеңЁжӯЈж•ҙж•°$N_1$пјҢдҪҝеҫ—еҪ“$n>N_1$ж—¶пјҢжңү$b_n < x$пјҢеҸ–$n>\max(N, N_1)$ж—¶пјҢжңү

иҝҷдёҺ$a^*$зҡ„е®ҡд№үзҹӣзӣҫгҖӮ

пјҲ3пјүд№ҹеҸҜд»ҘдёҚйҮҮз”ЁеҸҚиҜҒжі•гҖӮз”ұ$a^*$зҡ„жҖ§иҙЁеҸҜзҹҘпјҢеҸҜд»Һ$\{a_n\}$дёӯеҸ–еҮәдёҖдёӘеӯҗеҲ—$\{a_{k_n}\}$пјҢдҪҝеҫ—$\lim \limits_{n \to \infty}a_{k_n} = a^*$пјҢеҸҲеҪ“$k_n \ge n > N$пјҢ$a_{k_n} \le b_{k_n}$пјҢиҝҷж—¶еҸҜд»Ҙд»Һ$b_{k_n}$дёӯеҸ–еҮәдёҖдёӘж”¶ж•ӣеӯҗеҲ—$\{b_{i_{k_{n}}}\}$пјҢзӣёеә”зҡ„$\{a_{k_n}\}$зҡ„еӯҗеҲ—дёә$\{a_{i_{k_{n}}}\}$пјҢз”ұ$\lim \limits_{n \to \infty}a_{k_n} = a^*$пјҢеҸҜеҫ—

еҸҲз”ұ$a_{i_{k_{n}}} \le b_{i_{k_{n}}}$пјҢеҸҜеҫ—

иҖҢ$\{b_{i_{k_{n}}}\}$жҳҫз„¶д№ҹжҳҜ$\{b_n\}$зҡ„еӯҗеҲ—пјҢд»ҺиҖҢжңү

Q.E.D.

е®ҡзҗҶ4

еҜ№ж•°еҲ—$\{a_n\}$пјҢе®ҡд№ү$\alpha_n=\inf \limits_{k \ge n}a_kпјҢ\beta_n = \sup \limits_{k \ge n}a_k$пјҢйӮЈд№Ҳ

пјҲ1пјү$\{\alpha_n\}$жҳҜйҖ’еўһж•°еҲ—пјҢ$\{\beta_n\}$жҳҜйҖ’еҮҸж•°еҲ—пјӣ

пјҲ2пјү$\lim \limits_{n \to \infty} \alpha_n = a_*пјҢ\lim \limits_{n \to\infty} \beta_n =a^*$гҖӮ

иҜҒпјҡпјҲ1пјүжҢүе®ҡд№үпјҢжңү$\alpha_n = \inf\{a_n,a_{n+1},\cdots\}, \quad \alpha_{n+1}=\inf\{a_{n+1}, a_{n+2}, \cdots\}$пјҢжҳҫз„¶жңү$\alpha_n \le \alpha_{n+1}$пјҢеҗҢзҗҶеҸҜеҫ—$\beta_n \ge \beta_{n+1}$гҖӮ
пјҲ2пјүиҝҷйҮҢеҸӘиҜҒжҳҺеҸіиҫ№зӯүејҸпјҢе·Ұиҫ№зӯүејҸзҡ„иҜҒжҳҺжҳҜзұ»дјјзҡ„гҖӮ
пјҲaпјүе…Ҳи®ҫ$a^*$жҳҜдёҖдёӘжңүйҷҗж•°пјҢз”ұе®ҡзҗҶ1пјҲ1пјүеҸҜзҹҘ$a^* \in E$пјҢжүҖд»ҘеӯҳеңЁ$\{a_n\}$зҡ„дёҖдёӘеӯҗеҲ—$\{a_{k_n}\}$пјҢдҪҝеҫ—$\lim \limits_{n \to \infty}a_{k_n} = a^*$пјҢиҖҢ$k_n \ge n$пјҢжүҖд»Ҙ$a_{k_n} \le \sup\{a_n,a_{n+1},\cdots\}=\beta_n$пјҢд»ҺиҖҢжңү

еҜ№д»»ж„Ҹзҡ„$\varepsilon>0$пјҢз”ұе®ҡзҗҶ1пјҲ2пјүеҸҜзҹҘпјҢеӯҳеңЁ$n_0 \in N^*$пјҢеҪ“$n \ge n_0$ж—¶пјҢжңү$a_n \le a^* + \varepsilon$пјҢеӣ жӯӨ$\sup \limits_{k \ge n_0}a_n \le a^* + \varepsilon$пјҢеҚі$\beta_{n_0} \le a^* + \varepsilon$
пјҢеҸҲз”ұдәҺ$\{\beta_n\}$жҳҜйҖ’еҮҸзҡ„пјҢжүҖд»ҘеҪ“$n>n_0$ж—¶пјҢ$\beta_n \le \beta_{n_0} \le a^* + \varepsilon$пјҢеҶҚд»Ө$\varepsilon \to 0$пјҢд»ҺиҖҢжңү$\lim \limits_{n \to \infty}\beta_n \le a^* + \varepsilon$пјҢеҫ—

з”ұпјҲ1пјүпјҲ2пјүеҸҜеҫ—$\lim \limits_{n \to \infty}\beta_n = a^*$
пјҲbпјүи®ҫ$a^* = +\infty$пјҢеҲҷжңүдёҖеӯҗеҲ—и¶ӢдәҺ$+\infty$пјҢд»ҺиҖҢжңү

д»ҺиҖҢжңү$\lim \limits_{n \to \infty}\beta_n = +\infty$гҖӮ
