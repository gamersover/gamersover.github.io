leetcode题解130：被围绕的区域

发表于 2023-02-05 分类于 算法 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

描述

该题来自于力扣第130题

分析

要求的是被X围绕的区域，边界的不算。正面求解不好求，那就反面求解，即求出所有边界的O。

思路就很明了了，找出边界的O，然后从这些O开始深度优先遍历，找到所有相邻的O，可以标记下找出的O，那么最后将这些没有标记的O转为X就可以了。

深度优先遍历就很简单了，找出所有相邻的O，继续递归遍历即可。
